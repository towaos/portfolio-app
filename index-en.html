RaidersCTF.2025

System Overview

Authentication uses Cognito, and DynamoDB manages user and challenge information. Lambda handles authentication and challenge server startup/shutdown, while StepFunctions manages challenge server uptime. ECS launches tasks from images stored in ECR.

Challenges Addressed

  • Cost optimization by setting LunchTime to 15 minutes
  • Creating challenges across multiple fields using Docker only

  • Mobile responsive design
