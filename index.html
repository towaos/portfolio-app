RaidersCTF.2025

Next.js Go Cognito DynamoDB S3 Lambda StepFunction ECR/ECS Amplify Route53

システム概要

認証は Cognito を使用し、 DynamoDB でユーザー情報と問題情報を管理しています。 Lambda は認証や問題サーバーの起動・終了処理を担当し、 StepFunctions で問題サーバーの起動時間を管理します。 ECS は ECR に保存した イメージからタスクを起動します。

取り組んだ課題

  • 認証を簡易的にする為にsignin/signupそれぞれ名前とパスワードのみ
  • コスト最適の為にLunchTime15分に設定
  • Dockerのみで複数分野の問題作成
  • DynamoDBの特性を活かした管理
  • モバイルレスポンシブ対応

HexarkALG.2026

Next.js Go Cognito DynamoDB S3 Lambda StepFunction ECR/ECS Amplify Route53

システム概要

認証はCognitoを使用し、 DynamoDBでユーザー情報と問題情報を管理しています。 S3へのコード保存をトリガーに Lambdaが実行されECSが起動・コード実行&結果処理を行います。 StepFunctionsでWebEditerの起動時間を管理します。 ECSはECRに保存した イメージからタスクを起動します。

取り組んだ課題

  • 認証を簡易的にする為にsignin/signupそれぞれ名前とパスワードのみ
  • コスト最適の為にEditerTime60分に設定
  • EditerのCodeServerやLSPの利用
  • 複数言語のコード実行環境の作成
  • DynamoDBの特性を活かした管理
